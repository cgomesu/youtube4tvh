# Youtube Live M3U
Youtube Live M3U is a Python CLI program that extracts info from Youtube live-streams and creates/updates a M3U 
playlist that uses Streamlink (https://streamlink.github.io/) to feed any IPTV player (e.g., VLC) 
or server (TVHeadend).
